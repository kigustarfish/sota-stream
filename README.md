# SotA Stream

Overlay for Shroud of the Avatar streams

# Set up
* Install Python (v 2.7.10 +)
* Install dependencies `pip install -r requirements.txt`
* Set up config
* Run with python sota-stream.py
* Direct your browser to 127.0.0.1:8080 (this can also be reached by using your computer's IP from other devices)

# Config

* copy config.yml.example into config.yml
* follow instructions in comments for config.yml
